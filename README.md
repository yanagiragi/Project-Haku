# Project-Haku

Waifu Simulator

<br />

## <img src=./img/logo.png width=50px> <span style="margin: 10px">From Developers:</span>
    
    這是我們 MPY 三人在 國立交通大學 虛擬實境與擴充實境 做出來的作品。
    
    此 Repo 僅用來記錄一些與 Project Haku 有關的事物。

    這是我們第一次嘗試做 Vive 的作品，希望您會喜歡。

    如果對遊戲有興趣想要遊玩，請寄信給 此 Repo 的作者 :)
